Softboiled Glitch

This is a glitch that exists in Pokémon Diamond, Pearl, and Platinum. By using Softboiled or Milk Drink on an injured Pokemon, then performing a frame perfect directional input immediately after pressing A, you can heal something you shouldn't be able to. This lets you:

  • Revive a fainted Pokémon (normally not possible)
  • Overheal a Pokémon (basically cosmetic, as it gets fixed when the Pokémon takes damage in battle)
  • Heal an egg
  • Crash the game by trying to heal the Back button

Sometimes while overhealing a Pokémon the game will enter a seemingly endless heal loop that prevents you from doing any inputs. This is caused by an HP overflow.
