Japanese HeartGold/SoulSilver Slot Machines

Pokémon HeartGold and Soulsilver feature a slot machine minigame that is exclusive to the Japanese release. This minigame is playable at either the Goldenrod Game Corner or the Celadon Game Corner.

Predetermined Outcome

A hidden value between 0 and 6 is randomly generated every reel spin. This value corresponds to what you are capable of winning. If the game rolls a 0, the most common outcome, you cannot win anything from the slots. Even if the game rigs the first two reels, it will forcibly slip winning combinations off the third reel.

1, 2, 3, and 4 correspond to Replay, Marill, Pikachu, and Apricorns respectively. If you roll one of these, the game will usually rig the 1st and 2nd reels towards that outcome, but not always. If the first two reels are rigged, the third one is guaranteed to complete the sequence and result in a win.

If you roll a 5 or 6, you can hit the jackpot. Even if the game decides you can win the jackpot and it rigs the first two reels to be Poké Balls or 7’s, it will never rig the 3rd wheel to be a success. You must actually time your input to win the jackpot.

Mode

The slot machine minigame features three modes: Normal, Happy, and Slump. Every game starts in Normal mode. The current background color behind Smeargle shows the mode you are currently in. Green means you are in normal mode, red means you are in Happy mode, and blue means you are in Slump mode.

Difficulty

Upon starting the slot machine minigame, the game determines the difficulty of your session. These difficulties range from 0 to 3, with 0 being the hardest and 3 being the easiest. Difficulties 0, 1, or 2 are randomly selected upon starting on every machine except for the machine played by the Gentleman in the Goldenrod Game Corner. On Sunday mornings the Gentleman leaves this machine and it becomes accessible. This machine is always set to difficulty 3.
